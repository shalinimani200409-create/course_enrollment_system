# 📚 Course Enrollment System

A web-based Course Enrollment System developed using **Python (Flask)**, **HTML**, **CSS**, **JavaScript**, and **MySQL**. This application allows students to browse available courses, register, log in, and enroll in courses through a simple and user-friendly interface.

---

## 🚀 Features

- 🏠 Home Page
- 👨‍🎓 Student Registration
- 🔐 Student Login
- 📚 View Available Courses
- ✅ Course Enrollment
- 📂 Course Categories
- 📞 Contact Page
- 📊 Dashboard
- 💾 MySQL Database Integration
- 🎨 Responsive User Interface

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/course_enrollment_system.git
```

### 2. Open the Project Folder

```bash
cd course_enrollment_system
```

### 3. Create a Virtual Environment

```bash
python -m venv venv
```

### 4. Activate the Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### 5. Install Dependencies

```bash
pip install -r requirements.txt
```

### 6. Import Database

- Open **phpMyAdmin**
- Create a database
- Import the **database.sql** file

### 7. Update Database Configuration

Configure your MySQL credentials inside **app.py** (or your configuration file).

### 8. Run the Application

```bash
python app.py
```

### 9. Open in Browser

```
http://127.0.0.1:5000
```
